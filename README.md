# 🛒 Supermarket Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI Dashboard built to analyze supermarket sales data across different branches. The dashboard provides key business insights into revenue, profit margins, product line performance, and customer purchasing behavior to help managers make data-driven decisions.

## 💡 Key Business Insights from the Analysis
Based on the dashboard analysis of the dataset:
1. High Performing Categories: Food & Beverages and Sports & Travel generate the highest transaction volumes and ratings.
2. Payment Preferences: Digital wallets (Ewallets) and Cash are the most preferred payment methods, closely followed by Credit Cards.
3. Branch Performance: Branch A (Yangon) and Branch C (Naypyitaw) show strong performance, while Branch B (Mandalay) has areas for potential growth in customer acquisition.
4. Customer Demographics: Female customers contribute significantly to Health & Beauty, while Male customers show higher purchasing frequency in Electronic Accessories.

##  Tools & Technologies Used
- BI Tool: Power BI Desktop (Data Visualization & Dashboard Design)
- Data Source: Supermarket Sales Dataset (CSV format )
- Data Transformation: Power Query (for data cleaning, type conversions, and modeling)
